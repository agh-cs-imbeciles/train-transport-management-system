## Instructions how to run
1. Clone the repository
    ```
    git@github.com:agh-cs-imbeciles/train-transport-management-system.git
    ```
2. Go the _app_ directory
    ```
    cd app
    ```
3. Install node packages
    ```
    npm install
    ```
4. Run development server
    ```
    npm run dev
    ```
